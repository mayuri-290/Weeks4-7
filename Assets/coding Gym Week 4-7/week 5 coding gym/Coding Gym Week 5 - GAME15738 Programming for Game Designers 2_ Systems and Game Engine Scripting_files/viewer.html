Challenge # Description Difficulty
Reactions
1 Make a player that moves with keyboard input.
2
Give the player a world-space canvas, with its size and scale set so that it looks like a speech bubble over
the player’s head. Make the canvas a child of the player, so that it moves with the player. Put one Image on
the canvas, and set the anchors so that it stretches to fill the canvas.
3 Put an item or NPC in the scene. When the player is near them, show the canvas with a random reaction sprite.
Hide the canvas when the player moves away.
Controls
4 Make a screen-space canvas. Add a button, make the button change the colour of a sprite in the scene to a
random colour.
5 Add a slider to the UI, make the slider value change the rotation of the sprite.
6 Add sound effects when the button is clicked and the slider changes value.
Tape Deck
7 Make a screen-space UI with a background image and some text. And a play button that starts music playing from
an AudioSource, and a stop button that stops the music playing.
8 Make the tape heads rotate while the music is playing.
9 Use a slider to show the time remaining in the music as it is playing: set the max value of the slider to the
AudioClip length, update the value when the clip is playing.
10 Make a dropdown that lets you change the track.
Tab System
11 Make a screen-space canvas. Make a child game object to be the tabs, and make three buttons as children of the
tabs game object. Position the buttons to be tab 1, tab 2, and tab 3.
11
Make three more children of the canvas (not the tabs game object!) to be the tab panels, put an Image on each.
Write a script that has three public functions, one each for the three buttons. When button 1 is pressed, show
tab panel 1 and hide tab panels 2 & 3. When button 2 is pressed, show tab panel 2 and hide tab panels 1 & 3.
When button 3 is pressed, show tab panel 3 and hide tab panels 1 & 2.
12 Play a sound effect when the tab changes.
